Vec4f Class Reference

Detailed Description

Vector of 4 floats.

The Vec4f structure is basically made of four consecutive floats than can be adressed indifferently as:

  • an array of four floats
  • (x, y, z, w)
  • (dx, dy, dz, dw)
  • (r, g, b, a)
  • (red,green,blue,alpha)
  • (min, max, major, minor)
  • (width, height, depth)
  • (up, down, left, right)

Note that Vec4f is using several different names depending on the context

 typedef Vec4f Position;
 typedef Vec4f Size;
 typedef Vec4f Color;
 typedef Vec4f Orientation;
 typedef Vec4f Margin;
 typedef Vec4f Gravity;
 typedef Vec4f Range;
 typedef Vec4f Shape;
Examples:

glut-bash.cc, glut-cnft.cc, glut-colorbars.cc, and glut-terminal.cc.
