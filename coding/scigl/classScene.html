Scene Class Reference

Detailed Description

Container for objects and widgets.

A scene describes a set of objects and widgets and manage the view (zoom and orientation). Rendering of elements is done in the following order:

  1. Widgets with a negative z
  2. Objects
  3. Widgets with a positive z

Examples:

glut-axis.cc, glut-bash.cc, glut-cloud.cc, glut-cnft.cc, glut-cube.cc, glut-frame.cc, glut-image.cc, glut-line.cc, glut-scene.cc, glut-surface.cc, glut-texture.cc, and glut-viewer.cc.
