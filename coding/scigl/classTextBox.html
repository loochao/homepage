TextBox Class Reference

Detailed Description

TextBox rendering widget.

TextBox widget is used to display lines of formated text.

Examples:

glut-cnft.cc, and glut-text.cc.

Set size in window space.

It is possible to use relative size using coordinates between 0 and 1. Negative sizes relates to the complement of the related dimension. For example, if x is -10, then x size will be window_width-10. If x is -.25, then x size will be (1-.25)*window_width.
