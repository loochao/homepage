Colormap Class Reference

Detailed Description

A colormap represents a value range using colors.

A colormap is made of couple of (value,colors) where value range from min_ to max_ Any value between min_ and max_ is then linearly interpolated to get the corresponding color. The scale of the colormap can be changed using the scale method that allows to represent an arbitrary range. Any value above maximum value will be associated with the color of max. Any value below minimum value will be associated with the color of min.
